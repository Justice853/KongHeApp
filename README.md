# 空盒App
这是一款多功能工具类应用,《空盒》体积小巧，功能完善强大，且一直在不断完善中！！

APP特性：1.不足照片的体积，集成各种工具。 安装一个空盒，它将使用渺小的体积为您的设备提供各种不同的功能服务。

2.极快响应的启动速度大大提升了使用效率，启动后首页便是收藏，且不会显示任何杂乱的东西。

功能包括：翻译，天气查询，图片压缩，指南针，归属地查询，二维码制作等多种功能。。。。(*^_^*)
